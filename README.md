
# 🔐 Session Hijacking Simulator

A Python-based GUI tool that demonstrates **Broken Authentication vs Secure Session Management** through real-time attack simulations.

---

## 🚀 Features

* Toggle **Vulnerable** / **Secure** mode
* Simulated login & session handling
* Attack demos:

  * Prediction Attack
  * Session Sniffing
  * Sidejacking
  * Brute Force
  * Timeout Exploit
* Live logging dashboard
* Export logs to Excel

---

## 🛠️ Tech Stack

* Python
* Tkinter
* openpyxl

---

## ▶️ Run the Project

```bash
pip install openpyxl
python session_simulator.py
```

---

## 📚 What It Shows

* Weak vs strong session IDs
* Impact of broken authentication
* How security controls block attacks
